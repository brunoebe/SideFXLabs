""" Attach a numbered background image to the selected items in the network. """

This tool attaches numbers as background images to the selected nodes.
This is useful for instance when you want to highlight important steps in a network.


== How to use ==

# Select the nodes you want to attach images in desired order.

# Click on the shelf icon.
            
# The tool will automatically create quickmarks and background images attached to the selected nodes, in the selection order.

# You can navigate them clicking the shelf button with the following modifiers:

~~
:col:
      ((LMB))
:col:
      Number items and create quickmarks
:col:
:col:
~~
:col:
      ((LMB)) + ((CTRL))
:col:
      Jump to previous quickmark
:col:
:col:
~~
:col:
      ((LMB)) + ((ALT))
:col:
      Jump to next quickmark
:col:
:col:
~~
:col:
      ((LMB)) + ((SHIFT)) + ((CTRL))
:col:
      Jump to first quickmark
:col:
:col:
~~
:col:
      ((LMB)) + ((SHIFT)) + ((ALT))
:col:
      Jump to last quickmark
